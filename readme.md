A do it yourself python web application framework.
--

Setup:
- setup venv
- `pip install -r requirements.txt`

It is built on:
- Werkzeug
- SQLAlchemy
- jsonpickle
- alembic
- Faker
- pytest


how to install?
1. create a database, and put database details on config/config.py
